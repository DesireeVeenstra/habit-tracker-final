# Habit Tracker

A simple app to track daily habits with Firebase, AI chatbot, and biometric login.

## Features
- Add/edit/delete habits
- Persistent storage via Firebase
- AI suggestions
- Biometric security

## Setup
1. Clone the repo
2. Add Firebase config to `firebase.js`
3. Open `index.html`
